라이브러리들을 모아놓은것이 패키지

패키지들을 모아놓은것이 모듈

모듈들을 모아놓은것이 라이브러리(?)

*프레임워크는 특정 규약대로 가야한다. 흐름이 존재
 복잡한 문제를 해결하거나 서술하는데 사용되는 기본 구조
 웹페이지를 개발하는데 나타나는 어려움을 줄이는 것이 주목적으로 대표적으로 디비연동,
 템플릿 형태 표준, 세션 관리, 코드 재사용 기능 등의 기능을 포함하고 있다.

스프링 : 자바로 만들어진 프레임워크
장고 : 파이썬으로 만들어진 프레임워크
JSP : 자바 서버 페이지
스프링 부트 : 스프링의 세팅이 너무 어렵고 오래 걸리기 때문에 쉽게 만든 것

의존성 & 의존성 주입 알아오기!!

메이븐은 자바 프로젝트 빌드를 자동화 해주는 빌드 툴이다.
즉, 자바 소스를 컴파일하고, 패키지해서 deploy하는 일을 자동화 해주는 것

빌드란? 번역된 기계어를 컴퓨터에서 실행 가능한 파일로 만드는것.
컴파일 : 소스코드를 기계어로 번역

REST API : 웹설계의 우수성을 제대로 사용하기위해 사용하기 위한 아키텍쳐
Redis : 오픈소소기반의 비관계형 디비관리 시스템

쿠키는 클라이언트에 저장되는 키와 값이 들어 있는 작은 데이터 파일
세션은 서버 메모리에 저장되는 정보

JPA : 에플리케이션과 JDBC 사이에서 동작하는 도구
JDBC : 자바에서 데이터베이스를 접속할 수 있도록 하는 자바 인터페이스
